## Comandos Basicos do PostgreSQL

### iniciar servidor

```bash
sudo service postgresql start
```

### verificar status do servidor

```bash
sudo service postgresql status
```

### parar servidor

```bash
sudo service postgresql stop
```

### executar utilitário psql

```bash
sudo -u postgres psql
```
